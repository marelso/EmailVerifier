# Email Verifier

A command-line tool for verifying email domain configuration.

## Installation

Clone the repository:

~~~javascript
git clone https://github.com/yourusername/email-verifier.git
~~~

Install dependencies:

~~~javascript
cd email-verifier
go mod download
~~~
